Chatbot: Michael Afton

In this project, my team and I developed a chatbot designed to emulate the personality and speech patterns of Michael Afton from Five Nights at Freddy's (FNAF). Using the OpenAI API, we created an interactive chatbot capable of engaging in conversations with users, channeling Michael Afton’s eerie and cryptic style. The project combined HTML and CSS for the user interface with the OpenAI API for the chatbot's intelligent responses, along with additional libraries and tools to enhance the overall experience.


Technologies Used:

  • OpenAI API: For generating intelligent, personality-driven responses
  • HTML: The core structure for creating the chatbot’s interface
  • CSS: For styling the chatbot and creating a visually appealing user experience
  • JavaScript: For handling user input and integrating with the OpenAI API
  • GitHub: For version control and project collaboration
